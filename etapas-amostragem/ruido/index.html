🚆 Monitoramento de passagem de composição
Som residual — CURTO PERÍODO:
✓
Pelo menos 15 min de medição (contínua ou não)
✓
Monitoramento antes ou após a passagem da composição
Som residual — LONGA DURAÇÃO:
✓
Instalar o equipamento para monitoramento de longa duração
✓
Diurno: pelo menos 60 min de medição (contínua ou não)
✓
Noturno: pelo menos 30 min de medição (contínua ou não)
Som da passagem ferroviária:
✓
Considerar todo o tempo da passagem ferroviária
✓
Mínimo de 3 passagens monitoradas
✓
Pelo menos 1 passagem em cada sentido
✓
Se densidade ≤ 3 composições/dia: medir pelo menos uma passagem
✓
⚠️ NÃO realizar medições durante cruzamento de composições em linha dupla (mesmo que uma esteja parada)
✓
Sirenes, sinos, buzinas e campainhas de passagem em nível = sons intrusivos
✓
Anotar as características das composições monitoradas (tipo)

🚧 Operações em pátios / manobras / cruzamentos
✓
Sugerido: monitoramento 24 horas
Manobras:
✓

✓

Composição parada:
✓
Medir sem interferência de outras composições, cruzamentos, ultrapassagens ou manobras

✓

✓

✓
Pelo menos dois eventos de cruzamento ou ultrapassagem (diurno ou noturno)
✓
Mínimo de 30 eventos de operações de engates em período entre 60 min e 240 min — registrar LAFmax de cada evento
